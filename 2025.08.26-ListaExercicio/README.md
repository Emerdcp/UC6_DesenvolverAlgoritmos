## Exemplo de como fazer 

Um psidocodigo de como fazer.

```
programa {
  funcao inicio() {
    inteiro n1 = 0, n2 = 0, total = 0
    escreva("Informe um número: ")
    leia(n1)
    escreva("Informe outro número: ")
    leia(n2)
    total = n1 + n2
    escreva("O Resultado da Soma dos Dois é: " + total)
  }
}
```
